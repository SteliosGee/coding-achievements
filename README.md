# Coding Achievements

Gamify your coding journey. Track your habits, unlock achievements, and watch your progress grow — all from a sidebar panel in VS Code.

![VS Code](https://img.shields.io/badge/VS%20Code-1.85+-007ACC?logo=visual-studio-code&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)
![Version](https://img.shields.io/badge/Version-0.0.4-blue)

## How It Works

Code normally. The extension runs silently in the background, tracking your activity and unlocking achievements automatically. Open the sidebar panel anytime to see your progress.

## Features

- **Tier-colored icons** — locked achievements show a tint based on rarity (bronze, silver, gold, diamond)
- **Sound effects** — optional chime on unlock
- **Toast notifications** — popup when an achievement unlocks
- **Persistent progress** — survives extension updates and VS Code restarts
- **Collapsible categories** — Progress, Coding Habits, Time & Streaks, Files & Docs, Git & Debug
- **Multiple views** — Category, Tier, or Alphabetical
- **Responsive layout** — works at any panel width
- **Reset** — start over from the sidebar

## Getting Started

1. Install from the VS Code Marketplace
2. Open the sidebar panel (trophy icon in the activity bar)
3. Code — achievements unlock automatically

No configuration needed.

## Requirements

- VS Code 1.85 or higher

## Commands

| Command | Description |
|---|---|
| `Coding Achievements: Activate` | Manually trigger achievement check |
| `Coding Achievements: Reset` | Clear all progress |

## Privacy

All data is stored locally in VS Code's globalState. Nothing is transmitted externally.
